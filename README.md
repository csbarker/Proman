##Project Management App (PROMAN) 
PROMAN is a minimal project management system built in Ruby on Rails.

###Design
* The design selected for use is [Market Templates (Homepage)](https://dribbble.com/shots/1598629-Market-Templates-with-SideBar-Freebies) by [Brice Séraphin](http://briceseraphin.fr/).

###Features:
The [MoSCoW](http://en.wikipedia.org/wiki/MoSCoW_method) breakdown for v1 is as follows:
- **Must have**
 * Unlimited users
 * Unlimited projects
 * Unlimited tasks
- **Should have**
 * Time tracking
- **Could have**
 * Social logins
- **Won't have**
 * ???
